    Google Drive Setup

    Sudah otomatis terhubung ke Google Drive. Kolom di bawah hanya untuk admin — user survey tidak perlu isi apapun di sini.

    Export & Backup

    Download semua data tersimpan sebagai file CSV. Bisa dibuka di Excel.

    Hapus data yang sudah di-sync dari penyimpanan lokal HP. Data di Google Drive tetap aman.

    Cara Setup Google Drive

    1. Buka script.google.com, klik New Project
    2. Hapus semua kode → paste script dari developer (lihat file GOOGLE_APPS_SCRIPT.gs)
    3. Klik Run → setupInitial → izinkan akses ke Drive kamu
    4. Klik Deploy → New Deployment
      Type: Web App | Execute as: Me | Access: Anyone
    5. Copy URL → paste di kolom atas → Simpan & Test Koneksi
